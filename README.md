# Healthify - Mini Healthcare Support Web App

## Project Overview
Healthify is a simple healthcare support web application designed for NGOs to manage patient queries and volunteer registrations.

## Features
- Patient Support Form
- Volunteer Registration Form
- AI-based FAQ Chatbot (Rule-based)
- Automatic Priority Detection for Senior Citizens
- Auto-generated Submission Summary

## AI / Automation Idea
1. Rule-based chatbot for answering frequently asked healthcare queries.
2. Automatic priority tagging for patients aged above 60 (Senior Citizens).
3. Instant data summary after form submission.

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)
- Hosted on Vercel

## NGO Use Case
This app can help NGOs:
- Collect patient health concerns
- Identify high-priority patients
- Register volunteers easily
- Provide automated responses to common queries
